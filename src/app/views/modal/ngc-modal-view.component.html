Modal instance

const modalInstance = this.modalService.open(ModalContentComponent, ModalFooterComponent, ModalHeaderComponent);

Modal content components

Modal open method has 3 arguments:

modalService.open(contentModal: ComponentClassName, footerModal: ComponentClassName, modalOptions: ModalOptions

You can create every component as a normal Angular component and pass component's exported class name to the modalService.open(ModalContentComponent)



Close Modal

Every modal content can have a close method that will trigger modal's close method with a text argument

ContentModal.html

<button (click)="close('close modal')">Close Modal</button>

And where you save your modal's instance, you can subscribe to close method to know and act when close method has been triggered
